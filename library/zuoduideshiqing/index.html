做对的事情

定义与起源

"做对的事情"这一概念最早可追溯到段永平大学时期。他在2016年浙大60周年专访中回忆：

大三的时候，也不知道是我在哪看到，还是谁说的一句话，叫作"做对的事情，把事情做对"。 ——2016年 浙大60周年专访

段永平就读浙江大学无线电系，大三约在1981-1982年间。这句话对他"一生影响非常大"，后来成为步步高、OPPO、vivo等企业的核心经营哲学。

核心要义

1. 做对的事情 = 方向判断 = "道"

做对的事情是战略层面的判断，解决"做什么"和"不做什么"的问题。段永平将其比作"道"，而把事情做对则是"术"。

我承认这种现象的确存在，但这也只是短期表象。……一般来讲，聪明人似乎都知道如何把事情做对，但有智慧指的是要做对的事情。 ——2003年 北大总裁班演讲

2. 错的事情要马上停止

如果是错的事情，应该马上停止，对的事情我们要把事情做对。把事情做对的过程当中，犯错误是避免不了的，一定会犯错，所以有学习曲线，继续往前走就好了。 ——2025年 与王石对谈

3. 坚持做对的事情，而不是坚持做错的事情

人们常说的"坚持到底"，指的是坚持"做对的事情"而不是坚持做错的事情！ ——投资问答录

4. 在投资中，做对的事情 = 看懂公司未来现金流

所谓能看懂公司就是能看懂其未来现金流（做对的事情）。 ——投资问答录

实战案例

超级VCD诚信决策：2003年北大演讲中，段永平讲述了一个具体案例。步步高推出500线超级VCD时，竞争对手虚标800线。是否跟风虚标？段永平选择坚持真实标注500线，短期吃亏，但坚持了诚信这一"对的事情"。

离开小霸王****：1995年，段永平在小霸王做得很成功时选择离开，原因是"不开心，好多想法不能实现"。他答应原老板一年内不做同类产品，为守信赔钱做俄罗斯生意。这是选择"做对的事情"的典型案例。

投资网易****：段永平在2001-2002年网易股价低至不到1美元时大量买入，因为他看懂了游戏行业的未来。这是在投资领域"做对的事情"——看懂了公司未来现金流。

****步步高企业文化三十年：2025年与王石对谈中，段永平说："坚持做对的事情，发现错了赶紧改，几十年一直这样持续下来，它确实有很大威力。"步步高系企业三十年持续践行这一理念。

常见误解

误解一："做对的事情意味着永远不犯错"

段永平明确指出，把事情做对的过程中犯错是不可避免的。关键是方向（做对的事情）不能错，而执行中的错误可以容忍和纠正。

你要坚持什么？错的事情一定不能坚持，必须改；对的事情，要坚持。坚持的意思就是你在做对的事情，在把事情做对的过程当中，你会犯很多错误，会吃很多亏，这也是一个学习的过程。 ——2016年 浙大60周年专访

误解二："做对的事情是关于利益的判断"

段永平反复强调，做对的事情是关于"是非"而非"利益"的判断：

任何事情大部分人都会讨论有没有钱赚，我们会经常讨论，这是不是一件对的事情。 ——2025年 与王石对谈

误解三："最重要的是做了什么"

段永平纠正说，最重要的其实是不做什么：

最重要的要选择做对的事情，也就是发现错了的事情要改，不管多大的代价可能都是最小的代价。 ——2013-04-15

思想演变

1980年代：大学时期的初始感悟

大三时接触到"做对的事情，把事情做对"这句话，成为一生的思想基石。这一时期主要是个人层面的感悟。

1990-2000年代：企业经营的核心原则

在小霸王和步步高的经营中，"做对的事情"被转化为企业经营哲学。2003年北大演讲中，段永平用"聪明与智慧"的区别来阐释：聪明人知道如何把事情做对，有智慧的人知道要做对的事情。

2010年代：投资领域的延伸

段永平将"做对的事情"延伸到投资领域，等同于"看懂公司未来现金流"。同时与巴菲特的投资理念融合，形成了完整的投资哲学框架。

2020年代：与本分完全融合

2025年段永平将"做对的事情"定义为本分的核心内涵："本分就是做你该做的事情，就是理性、想长远。"做对的事情从一个独立的原则变成了本分文化的有机组成部分。
